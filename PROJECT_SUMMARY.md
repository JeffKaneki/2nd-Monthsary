## 🚀 Quick Start (5 minutes)

### Step 1: Open Website
```bash
Open index.html in your browser
# Or use local server: python -m http.server 8000
```

### Step 2: Add Your Date
Edit `script.js`, change line 10:
```javascript
RELATIONSHIP_START: new Date('2024-03-09'), // YOUR DATE HERE
```

### Step 3: Add Your Photo
1. Save couple photo as `couple-photo.jpg`
2. Place in: `assets/images/`
3. Done! (Automatically used in hero section)

### Step 4: Customize Text
Edit `index.html`:
- Change hero title/subtitle
- Update timeline events
- Edit reason cards
- Personalize love letter in `special.html`

### Step 5: Share!
Open `index.html` and share link with your love! 💜

## 🆘 Need Help?

### Common Questions
- **"How do I add photos?"** → See ASSETS_GUIDE.md
- **"How do I change the date?"** → Edit line 10 of script.js
- **"How do I add more reasons?"** → See CUSTOMIZATION_EXAMPLES.md
- **"How do I share online?"** → Upload to Netlify (free)
- **"Where do I add music?"** → Place in assets/music/, uncomment in script.js

### Troubleshooting
1. **Images not showing?**
   - Check file path in HTML
   - Verify images in assets/images/ folder
   - Clear browser cache

2. **Games not working?**
   - Refresh page
   - Clear cache
   - Try different browser
   - Check console (F12)

3. **Animations not smooth?**
   - Close browser tabs
   - Update browser
   - Disable extensions
   - Use Chrome for best performance

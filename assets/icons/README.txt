# 🎨 Icons Folder (Optional)

Custom icons and SVG graphics (optional enhancement)

## What Goes Here (Optional)

This folder is for optional custom icons:
- Custom SVG graphics
- Icon sets
- Custom emojis/stickers

## Note

**Emojis are already included!** 
The website uses Unicode emojis (❤️ 💜 💖 etc.) which don't require any image files.

## Optional SVG Icons

If you want custom SVG icons:

1. Create or download .svg files
2. Place them in this folder
3. Reference in HTML like:
   ```html
   <img src="assets/icons/heart.svg" alt="Heart" width="30">
   ```

## Example SVG Heart

Create **heart.svg**:
```xml
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 100 100">
  <path fill="#ff1493" d="M50,90 C50,90 20,70 20,50 C20,35 32,25 42,25 C50,25 50,35 50,35 C50,35 50,25 58,25 C68,25 80,35 80,50 C80,70 50,90 50,90 Z"/>
</svg>
```

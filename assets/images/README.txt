# 🖼️ Images Folder

Place your couple photos and memory images here!

## What Goes Here

1. **couple-photo.jpg** - Your couple photo for the hero section (300x300px recommended)
2. **memory1.jpg** through **memory6.jpg** - Your relationship memory photos (800x800px recommended)

## File Organization

```
images/
├── couple-photo.jpg      ← Hero section image
├── memory1.jpg          ← First memory photo
├── memory2.jpg          ← Second memory photo
├── memory3.jpg          ← Third memory photo
├── memory4.jpg          ← Fourth memory photo
├── memory5.jpg          ← Fifth memory photo
└── memory6.jpg          ← Sixth memory photo
```

## Tips

✅ Use high-quality photos
✅ Compress images with TinyPNG
✅ Use consistent dimensions
✅ Keep file sizes under 300KB each
✅ Use descriptive names

For detailed instructions, see: **ASSETS_GUIDE.md**
